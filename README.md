# MSAP_LV3_2024
LV3 - statističko zaključivanje
Ovaj repozitorij sadrži materijale i rješenja za laboratorijske vježbe iz područja statističkog zaključivanja. Glavni fokus je na analizi podataka, odabiru odgovarajućih statističkih metoda, vizualizaciji podataka i interpretaciji rezultata.


**Zadatak 1: Zaključivanje na malom uzorku**

Analizira se preciznost mjernog instrumenta pomoću mjerenja masa dvaju različitih listova papira. Glavni ciljevi uključuju:

- Testiranje jednakosti prosječnih masa na razinama značajnosti od 5% i 1%.
- Primjena odgovarajućih statističkih testova
- odgovarajuća interpretacija rezultata

**Zadatak 2: Parametarski/neparametarski testovi**

Analizira se učinkovitost programa za poboljšanje vještina sudionika. Glavni koraci uključuju:

- Provođenje deskriptivne statistike.
- Testiranje normalnosti podataka.
- Odabir odgovarajuće statističke metode
- Vizualizaciju podataka

Pisanje izvještaja o dobivenim rezultatima za oba zadatka
